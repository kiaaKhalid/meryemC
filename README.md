# Meryem: Industrial Predictive Maintenance

A professional AI-powered dashboard for industrial maintenance management in Casablanca.

## Technical Stack
- **Backend**: Spring Boot 3.x (Java)
- **Frontend**: React (Next.js) + Tailwind CSS
- **AI Service**: FastAPI (Python) with Keras & XGBoost

## Features
- Real-time asset tracking
- Weather-based risk prediction
- Executive KPI dashboard
- Equipment health diagnostics
